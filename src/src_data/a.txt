在智能家居管理场景下，用户请求构建一个页面来控制所有的家居，现在有以下vue组件
{
    名称：智能化容器
    模板：smart_container_pro
    用途：容纳同一类型或和服务于同一目的的不同组件，并以合理的方式组织他们，可用嵌套使用来将组件分组
    属性：title(string)
    属性可选项：{
        title：说明该组件组作用的文字
    }
    补充：尽量以比较细的分类将相同类型的组件放进相同的智能化容器里
}
{
    名称：连续数据滑块
    模板：smooth_slider
    用途：绑定数据名，自动从后端获取该数据，并确定一个合理的滑动上下限，在滑动时自动将其修改，也可以显示数据
    属性：data_name(string)
    属性可选项:{
        data_name:{
            洗衣机水量设定，
            卧室灯亮度，
            卧室空调温度，
            
            客厅灯亮度，
            客厅空调温度，
            浴室水温，
            洗衣机定时，
            猫粮投喂量，

        }
    }
}
{
    名称：开关按钮
    模板：switch_button
    用途：绑定后端布尔型数据，在点击时自动反转其状态（如果可以的话），并显示其状态
    属性：data_name(string)

    属性可选项:{
        data_name:{
            卧室灯开启，
            厨房灯开启，
            客厅灯开启，
            洗衣机开启，
            卧室空调开启，
            客厅空调开启，
            猫粮投喂器开启，
            卧室杀菌开启
        }
    }
}
请以标签的形式将组件组织起来，不需要给出完整的html结构，你生成的代码将被嵌入已有的网页文件
